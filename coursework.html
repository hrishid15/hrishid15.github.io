Coursework

Current

ECE 408 — Applied Parallel Programming

CUDA C/C++ • GPU Programming • Parallel Algorithms • Performance Optimization

Focuses on high-performance computing and GPU acceleration using CUDA C/C++. Students implement parallel algorithms for convolution, matrix multiplication, and reduction, learning to optimize memory access, synchronization, and shared-memory tiling.

Course Notes (In Progress!)

ECE 385 — Digital Systems Laboratory

SystemVerilog • FPGAs • Digital Logic Design • Hardware Debugging

Hands-on lab course in digital design using SystemVerilog and FPGAs. Covers finite state machines, pipelined architectures, and hardware debugging through projects such as building a 16-bit LC-3-style microprocessor from scratch.

ECE 374 — Algorithms & Models of Computation

Algorithm Analysis • Graph Theory • Dynamic Programming • Complexity Proofs

Studies algorithmic problem-solving and theoretical models of computation. Emphasizes proofs of correctness, asymptotic complexity, graph traversal, and dynamic programming across real-world computational problems.

Course Notes (In Progress!)

ECE 484 — Principles of Safe Autonomy

Control Theory • Formal Verification • Reachability Analysis • ROS/Gazebo

Introduces concepts for designing and verifying autonomous systems with safety guarantees. Topics include reachability analysis, control barrier functions, formal verification, and human-in-the-loop system design for air and ground vehicles.

Completed

ECE 391 — Computer Systems Engineering

C Programming • RISC-V Assembly • Operating Systems • Concurrency

Core systems course covering operating system internals, concurrency, and device drivers. Students build components like schedulers, file systems, and interrupt handlers while working directly with kernel-level C and RISC-V assembly.

ECE 479 — Internet of Things and Cognitive Computing

IoT Protocols • Edge Computing • Cloud Integration • Embedded Systems

Explores IoT architecture, edge intelligence, and cloud integration. Includes design of sensor networks, real-time data analytics, and adaptive systems using AI-driven decision-making for connected devices.

ECE 313 — Probability with Engineering Applications

Probability Theory • Random Variables • Statistical Modeling • Data Analysis

Covers fundamental probability theory and its use in stochastic modeling and data analysis. Topics include random variables, expectation, conditional probability, and statistical inference for engineering systems.

CS 225 — Data Structures

C++ • Object-Oriented Programming • Algorithm Efficiency • Memory Management

Focuses on algorithmic efficiency and implementation of abstract data types in C++. Covers trees, graphs, hash tables, and runtime analysis, culminating in large-scale projects emphasizing memory and performance optimization.
