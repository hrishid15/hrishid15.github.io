Key Features

  • Unix-style kernel supporting ELF loading, dynamic user stack construction, argument passing, and seamless supervisor/user mode transitions
  • VirtIO block device driver with descriptor-based I/O, interrupt-driven completions, and reusable queues with reset and locking logic for concurrency
  • Virtual memory management with Sv39 page tables, enabling per-process memory isolation, page-level protection, and user/kernel memory separation
  • Preemptive multithreading system with thread creation/joining, stack allocation, context switching, and safe trapframe management
  • Custom writable filesystem supporting file operations, hierarchical directories, bitmap-based block allocation, and page-cache dirty tracking for disk persistence
  • Minimal user-mode shell with argument parsing, file interaction, and concurrent process support via custom syscalls and inter-process pipes
